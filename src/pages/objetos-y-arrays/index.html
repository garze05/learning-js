Objetos y Arrays

| Los métodos de los objetos y arrays son de lo más usado en JavaScript.

Nombre del Curso Profesor Nota

Se pueden aplicar métodos como .forEach(), .find(), .some(), .every(), .map(), .filter(), .reduce(), etc.

El código para generar la tabla de arriba: 

              const tbodyCursos = document.getElementById("tbodyCursos");

              // Declaramos array de objetos de cursos
              const misCursos = [
                { nombre: "Virtualización", profesor: "Carlos Vega", nota: 90 },
                { nombre: "Redes", profesor: "Pablo Jose", nota: 85 },
                { nombre: "Lenguaje C#", profesor: "Fernando Ramirez", nota: 95 },
                { nombre: "Lenguaje de Consulta de Base de Datos", profesor: "Richard García", nota: 100 },
              ];

              // .forEach() ejecuta un bloque de código para cada elemento del array, sin modificar el array original
              misCursos.forEach((curso) => {
                const tr = document.createElement("tr"); // Creamos una fila para cada curso
                Object.values(curso).forEach((valor) => { // Para cada valor del objeto curso actual que estamos iterando
                  const td = document.createElement("td"); // Creamos una celda para cada valor
                  td.textContent = valor;
                  tr.append(td); // Añadimos la celda a la fila
                });

                tbodyCursos.append(tr); // Finalmente, añadimos la fila al cuerpo de la tabla
              });
